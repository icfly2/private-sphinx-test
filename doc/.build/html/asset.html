Asset¶

asset
The set of parameters defining the cost of energy production. These parameters can include the location, which determine subsidy payments, efficiencies and power levels as well stochastic parameters describing the overall performance of the energy asset, such as availability, willingness to participate in events or additional cost.

add_asset(name, data)[source]¶

Add a new asset by writing the information to assets.csv

Be sure to enter information in the correct units, and not use semi-colons in the asset description text. Also all required data tags must be supplied, else a KeyError will be raised, stating the missing keys.

Parameters:
  • name (string) – Name of the new asset, must be unique as it is
  • as index in the database. (used) –
  • data (dict) – The data and parameters of the asset.
Returns:

None
Raises:

KeyError – If required data wasn’t specified or an existing key was used to name a new asset.

Examples

Add an asset called example with these parameters: - Power: 10 MW - Description: This is an example asset. - Location: UK_SE - Efficiency: 0.33

>>> import flexibility as fl
>>> A = fl.asset.Asset()
>>> data ={
        'power':10,
        'description':'This is an example asset',
        'efficiency':0.33,
        'location':'UK_SE',
        'realised':False,
        }
>>> A.add_asset(name='example',data=data)
>>> A.all_assets #check the new asset is listed.
