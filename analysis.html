Analysis¶

The analysis module is used to analyse an asset. It loads the data and scheduling methods defined in the Data and Scheduling modules.

class flexibility.analysis.Analysis(**kwargs)[source]¶

Load historical data and analyse asset performance.

Optionally set new to the name of a predefined asset to define the asset on initialisation.

Parameters:
  • new (string) – Name of the new asset, which must already excisit in
  • database. Providing this name calls set_asset. (the) –

Examples

Create and set an asset as amp and send it for analysis:

>>> import flexibility as fl
>>> amp = fl.Asset('amp')
>>> a = Analysis(asset=amp)

Or do this in one operation:

>>> a = fl.Analysis(asset_name='amp')
